### Character Requirements

To ensure compatibility with our animation system, contributed characters should:

| Requirement | Details |
|-------------|---------|
| **Format** | GLTF/GLB format |
| **Rigging** | Compatible with Mixamo or similar |
| **Animations** | Walk, Run, Celebrate, Sit-ups/Exercise |
| **Style** | Low-poly, stylized (matching current art style) |
| **Content** | Family-friendly, no offensive imagery |

### How to Submit

1. **Fork** the [GitHub repo](https://github.com/sonyschan/TrencherRPG)
2. Add your character to `/public/assets/characters/`
3. Include a preview image
4. **Submit a Pull Request** with:
   - Character name
   - Brief description
   - Your credit name (or stay anonymous)

We'll review submissions and integrate approved characters into the game!
