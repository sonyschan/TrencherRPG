# 공유 가능한 OG 카드

IdleTrencher 포트폴리오를 스타일리시하게 공유하세요! 소셜 미디어에서 지갑 링크를 공유하면 아름다운 미리보기 카드가 자동으로 생성됩니다.

## OG 카드란?

OG(Open Graph) 카드는 Twitter/X, Discord, Telegram 등의 소셜 미디어 플랫폼에서 링크를 공유할 때 나타나는 미리보기 이미지입니다. IdleTrencher는 당신의 지갑을 위한 독특하고 눈길을 끄는 카드를 생성합니다.

## 사용 방법

1. **OG 카드 URL 얻기** - 지갑 주소 사용
2. **어디든 공유하기** - Twitter, Discord, Telegram 등
3. **마법을 지켜보세요** - 아름다운 미리보기 카드가 자동으로 나타납니다

### OG 이미지 URL

포트폴리오 OG 카드는 다음 URL에서 사용 가능합니다:
```
https://idletrencher.xyz/api/og/당신의_지갑_주소
```

예시:
```
https://idletrencher.xyz/api/og/9jwHJHSD7geYvTy6WUtoDVuuvuoJiWH2XHWMggPUpump
```

## 카드 특징

OG 카드에는 다음이 포함됩니다:

- **IdleTrencher 브랜드 디자인** - GM/GN 테마 아트워크
- **동적 지갑 미리보기** - 포트폴리오를 한눈에 표시
- **최적화된 크기** - 모든 주요 플랫폼 지원 (1200x630)
- **자동 생성** - 스크린샷 불필요

## 테마

| 테마 | 설명 |
|------|------|
| **GM (굿모닝)** | 밝고 에너지 넘치는 디자인, 낮 시간 공유에 적합 |
| **GN (굿나잇)** | 차분하고 편안한 디자인, 저녁 분위기에 적합 |
