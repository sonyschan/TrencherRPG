# 레벨 & EXP 시스템

파트너는 포트폴리오에 있는 것만으로 **경험치 (EXP)**를 얻습니다. 오래 보유할수록 더 강해집니다!

![EXP와 레벨](/img/partners-exp-level.jpg)

## 수학 공식

IdleTrencher는 투명하고 결정론적인 레벨링 공식을 사용합니다:

### EXP 계산

```
유효 EXP = 보유 시간(시간) + 보너스 시간
```

- **보유 시간**: 첫 지갑 연결 이후 시간 (1시간 = 1 EXP)
- **보너스 시간**: 가격 상승으로 인한 추가 EXP (적극적 시장 참여 보상)

### 레벨 공식

```
레벨 = floor(sqrt(시간 / 2.433))
```

이것은 **제곱근 진행**을 만듭니다 — 초기 레벨은 빨리 도달하고, 높은 레벨은 기하급수적으로 더 많은 시간이 필요합니다.

### 각 레벨에 필요한 시간

| 레벨 | 필요 시간 | 대략적 기간 |
|------|-----------|-------------|
| 1 | 0 | 즉시 |
| 2 | 10 | ~10시간 |
| 5 | 61 | ~2.5일 |
| 10 | 243 | ~10일 |
| 20 | 973 | ~40일 |
| 30 | 2,189 | ~3개월 |
| 40 | 3,893 | ~5.5개월 |
| 50 | 6,083 | ~8.5개월 |
| 60 | 8,759 | ~1년 (최대) |

### 역 공식 (레벨에서 시간 계산)

```
시간 = 레벨² × 2.433
```

## 레벨 마일스톤

| 레벨 | 필요 시간 |
|------|-----------|
| Lv2 | ~10시간 |
| Lv10 | ~10일 |
| Lv20 | ~40일 |
| Lv30 | ~3개월 |
| Lv60 | ~1년 (최대) |

## 보너스 EXP 시스템

가격 상승은 **보너스 시간**을 부여하여, 긍정적 시장 움직임을 경험한 홀더에게 보상합니다:

- 토큰 가격이 오르면 보너스 EXP가 축적됩니다
- 이것은 균형 잡힌 전략을 만듭니다: 꾸준한 보유로 기본 성장, 가격 상승을 잡아 가속 진행

:::warning EXP 리셋
파트너가 상위 보유에서 빠지면, EXP가 리셋됩니다. 이 설계는 균형 잡힌 포트폴리오와 일관된 보유 전략을 장려합니다.
:::

## 왜 이런 설계?

제곱근 공식이 보장하는 것:

1. **새 플레이어가 진행을 느낌** — 단 10시간으로 Lv2 도달
2. **장기 홀더가 보상받음** — 높은 레벨은 진정한 헌신을 보여줌
3. **최고 레벨 달성 가능** — 약 1년으로 Lv60은 현실적
4. **투명하고 검증 가능** — 누구나 예상 레벨을 계산 가능

> 천리 길도 1시간 보유에서 시작됩니다. 🧘
