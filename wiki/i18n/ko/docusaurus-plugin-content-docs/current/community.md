## 3D 캐릭터 기여

캐릭터 컬렉션을 확장하기 위해 **3D 아티스트**를 적극적으로 모집하고 있습니다!

### 왜 기여하나요?

✅ 당신의 작품이 게임의 일부가 됩니다
✅ 자동으로 자신의 스킨을 사용할 수 있습니다
✅ 게임과 Wiki에서 크레딧
✅ 재미있는 커뮤니티 프로젝트에 참여

### 캐릭터 요구 사항

애니메이션 시스템과의 호환성을 위해, 기여하는 캐릭터는:

| 요구 사항 | 세부 사항 |
|----------|----------|
| **포맷** | GLTF/GLB 포맷 |
| **리깅** | Mixamo 또는 유사 시스템과 호환 |
| **애니메이션** | 걷기, 달리기, 축하, 윗몸 일으키기/운동 |
| **스타일** | 로우폴리, 스타일라이즈드 (현재 아트 스타일에 맞춤) |
| **콘텐츠** | 전 연령 적합, 부적절한 이미지 없음 |

### 제출 방법

1. [GitHub 리포지토리](https://github.com/sonyschan/TrencherRPG)를 **Fork**
2. 캐릭터를 `/public/assets/characters/`에 추가
3. 프리뷰 이미지 포함
4. **Pull Request** 제출 시 포함:
   - 캐릭터 이름
   - 간단한 설명
   - 크레딧 이름 (또는 익명 선택)

제출물을 검토하고 승인된 캐릭터를 게임에 통합합니다!
