# 시작하기

몇 초 만에 지갑을 연결하고 IdleTrencher 여정을 시작하세요!

![지갑 연결](/img/wallet-connection.jpg)

## 지갑 연결

IdleTrencher는 **Privy** — 신뢰할 수 있는 서드파티 인증 시스템 — 을 사용하여 Solana 지갑을 안전하게 연결합니다. 이것은:

- **시드 문구 노출 없음** — 우리는 개인 키를 요청하거나 저장하지 않습니다
- **다중 지갑 지원** — Phantom, Solflare 또는 모든 Solana 지갑으로 연결 가능
- **원활한 경험** — 데스크톱과 모바일 모두 원클릭 연결

## 연결 방법

1. **지갑 연결** 버튼 클릭
2. Privy 모달에서 선호하는 지갑 선택
3. 지갑에서 연결 승인
4. 토큰이 파트너로 변신!

## 다음에 무슨 일이?

연결 후, IdleTrencher는 자동으로:

- 지갑의 토큰 보유량 스캔
- 가장 많이 보유한 토큰을 3D 파트너 캐릭터로 변환
- 포트폴리오 가치 계산
- 첫 방문부터 EXP 추적 시작

:::tip 모바일 사용자
IdleTrencher는 완전 모바일 지원! 스마트폰 브라우저에서 열고 모바일 지갑 앱에 연결하세요. 파트너들이 어디서든 기다리고 있어요.
:::

## 프라이버시 & 보안

- **퍼블릭 블록체인 데이터**만 읽습니다 — 블록 익스플로러에서 누구나 볼 수 있는 정보와 동일
- 개인 데이터 수집 없음
- 거래 활동 추적이나 분석 없음
- 언제든지 지갑 연결 해제 가능
