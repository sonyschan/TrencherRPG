# 开始使用

几秒内连接你的钱包，开始你的 IdleTrencher 旅程！

![钱包连接](/img/wallet-connection.jpg)

## 钱包连接

IdleTrencher 使用 **Privy** — 一个可信赖的第三方验证系统 — 来安全连接你的 Solana 钱包。这意味着：

- **不会暴露助记词** — 我们从不索取或存储你的私钥
- **支持多种钱包** — 可使用 Phantom、Solflare 或任何 Solana 钱包连接
- **无缝体验** — 桌面和手机都能一键连接

## 如何连接

1. 点击 **连接钱包** 按钮
2. 从 Privy 弹窗中选择你偏好的钱包
3. 在你的钱包中批准连接
4. 你的代币就会变成伙伴！

## 接下来会发生什么？

连接后，IdleTrencher 会自动：

- 扫描你的钱包中的代币持有
- 将你持有最多的代币转化为 3D 伙伴角色
- 计算你的投资组合价值
- 从你第一次造访开始追踪经验值

:::tip 手机用户
IdleTrencher 完全支持手机！在手机浏览器中打开并连接你的手机钱包应用。你的伙伴们会随时随地等着你。
:::

## 隐私与安全

- 我们只读取 **公开的区块链数据** — 和任何区块浏览器上能看到的信息一样
- 不收集个人数据
- 不追踪或分析你的交易活动
- 随时可以断开钱包连接
