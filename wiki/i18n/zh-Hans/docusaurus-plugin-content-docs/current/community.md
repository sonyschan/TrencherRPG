## 贡献 3D 角色

我们正在积极寻找 **3D 艺术家** 来帮助扩展我们的角色收藏！

### 为什么要贡献？

✅ 你的创作会成为游戏的一部分
✅ 你自动可以使用自己的皮肤
✅ 在游戏和 Wiki 中获得署名
✅ 加入一个有趣的社区项目

### 角色要求

为确保与我们的动画系统兼容，贡献的角色应该：

| 要求 | 详情 |
|------|------|
| **格式** | GLTF/GLB 格式 |
| **骨架** | 与 Mixamo 或类似系统兼容 |
| **动画** | 走路、跑步、庆祝、仰卧起坐/运动 |
| **风格** | 低多边形、风格化（符合目前的美术风格）|
| **内容** | 适合全年龄、无冒犯性图像 |

### 如何提交

1. **Fork** [GitHub 仓库](https://github.com/sonyschan/TrencherRPG)
2. 将你的角色添加到 `/public/assets/characters/`
3. 包含预览图片
4. **提交 Pull Request** 并附上：
   - 角色名称
   - 简短描述
   - 你的署名（或选择匿名）

我们会审核提交并将批准的角色整合到游戏中！
