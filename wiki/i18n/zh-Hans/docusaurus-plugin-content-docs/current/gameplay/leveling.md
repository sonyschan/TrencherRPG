# 等级与经验值系统

你的伙伴只要在你的投资组合中就会获得 **经验值 (EXP)**。持有越久，他们就越强！

![经验值和等级](/img/partners-exp-level.jpg)

## 数学公式

IdleTrencher 使用透明、确定性的升级公式：

### 经验值计算

```
有效经验值 = 持有小时数 + 奖励小时数
```

- **持有小时数**：从首次钱包连接以来的时间（1 小时 = 1 EXP）
- **奖励小时数**：价格上涨带来的额外经验值（奖励积极的市场参与）

### 等级公式

```
等级 = floor(sqrt(小时数 / 2.433))
```

这创造了 **平方根进度** — 早期等级来得很快，而更高等级需要指数级更多的时间。

### 每个等级所需时间

| 等级 | 所需小时数 | 大约时间 |
|------|------------|----------|
| 1 | 0 | 立即 |
| 2 | 10 | ~10 小时 |
| 5 | 61 | ~2.5 天 |
| 10 | 243 | ~10 天 |
| 20 | 973 | ~40 天 |
| 30 | 2,189 | ~3 个月 |
| 40 | 3,893 | ~5.5 个月 |
| 50 | 6,083 | ~8.5 个月 |
| 60 | 8,759 | ~1 年（最高）|

### 反向公式（从等级计算小时数）

```
小时数 = 等级² × 2.433
```

## 等级里程碑

| 等级 | 所需时间 |
|------|----------|
| Lv2 | ~10 小时 |
| Lv10 | ~10 天 |
| Lv20 | ~40 天 |
| Lv30 | ~3 个月 |
| Lv60 | ~1 年（最高）|

## 奖励经验值系统

价格上涨会给予 **奖励小时数**，奖励经历正向市场波动的持有者：

- 当代币价格上涨时，奖励经验值会累积
- 这创造了平衡策略：稳定持有获得基础成长，把握价格上涨获得加速进度

:::warning 经验值重置
如果伙伴从你的顶级持有中掉出，他们的经验值会重置。这个设计鼓励维持平衡的投资组合和一致的持有策略。
:::

## 为什么这样设计？

平方根公式确保：

1. **新玩家感受到进度** — 只要 10 小时就能达到 Lv2
2. **长期持有者获得奖励** — 更高等级展示真正的承诺
3. **最高等级可达成** — 约 1 年达到 Lv60 是现实的
4. **透明且可验证** — 任何人都能计算预期等级

> 千里之行，始于持有一小时。🧘
