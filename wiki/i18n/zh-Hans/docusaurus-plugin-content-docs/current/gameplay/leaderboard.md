## 终极炫耀

当然，任何人都可以买代币。但会交易、有眼光、能持有，三种能力加起来才是真功夫。排行榜将呈现出社区中代币修为最高的持有者。

## 排名机制

你的 **总分** 是通过加总所有伙伴角色的等级来计算：

```
总分 = 所有伙伴等级相加
```

例如：
- 伙伴 1：Lv15
- 伙伴 2：Lv12
- 伙伴 3：Lv8
- **总分：35**

## 联赛系统

不同资产等级在各自的联赛中竞争 — 确保公平竞争：

| 联赛 | 投资组合价值 |
|------|--------------|
| 铜牌 | < $1,000 |
| 银牌 | $1,000 - $10,000 |
| 金牌 | $10,000 - $100,000 |
| 钻石 | $100,000+ |

这样，小户和大户各有自己的竞争空间。

## 为什么要有排行榜？

说实话 — 加密货币可以很竞争。与其比谁交易最快或谁最早得到内线消息，我们想庆祝一种不同的成功：

**在持币与积极交易当中，取得平衡的能力。**

听起来很简单，但在一个充满百倍 meme 币和日常 rug pull 的世界里，成为一个稳定的持有者本身就是一种成就。

> 「股市是一个把钱从没耐心的人转移到有耐心的人的工具。」
> — 沃伦·巴菲特（他可能不懂 meme 币，但原则是对的）
