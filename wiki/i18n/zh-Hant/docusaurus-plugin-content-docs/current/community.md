## 貢獻 3D 角色

我們正在積極尋找 **3D 藝術家** 來幫助擴展我們的角色收藏！

### 為什麼要貢獻？

✅ 你的創作會成為遊戲的一部分
✅ 你自動可以使用自己的造型
✅ 在遊戲和 Wiki 中獲得署名
✅ 加入一個有趣的社群專案

### 角色要求

為確保與我們的動畫系統相容，貢獻的角色應該：

| 要求 | 詳情 |
|------|------|
| **格式** | GLTF/GLB 格式 |
| **骨架** | 與 Mixamo 或類似系統相容 |
| **動畫** | 走路、跑步、慶祝、仰臥起坐/運動 |
| **風格** | 低多邊形、風格化（符合目前的美術風格）|
| **內容** | 適合全年齡、無冒犯性圖像 |

### 如何提交

1. **Fork** [GitHub 倉庫](https://github.com/sonyschan/TrencherRPG)
2. 將你的角色添加到 `/public/assets/characters/`
3. 包含預覽圖片
4. **提交 Pull Request** 並附上：
   - 角色名稱
   - 簡短描述
   - 你的署名（或選擇匿名）

我們會審核提交並將批准的角色整合到遊戲中！
