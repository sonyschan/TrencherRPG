# 開始使用

幾秒內連接你的錢包，開始你的 IdleTrencher 旅程！

![錢包連接](/img/wallet-connection.jpg)

## 錢包連接

IdleTrencher 使用 **Privy** — 一個可信賴的第三方驗證系統 — 來安全連接你的 Solana 錢包。這意味著：

- **不會暴露助記詞** — 我們從不索取或儲存你的私鑰
- **支援多種錢包** — 可使用 Phantom、Solflare 或任何 Solana 錢包連接
- **無縫體驗** — 桌面和手機都能一鍵連接

## 如何連接

1. 點擊 **連接錢包** 按鈕
2. 從 Privy 彈窗中選擇你偏好的錢包
3. 在你的錢包中批准連接
4. 你的代幣就會變成夥伴！

## 接下來會發生什麼？

連接後，IdleTrencher 會自動：

- 掃描你的錢包中的代幣持有
- 將你持有最多的代幣轉化為 3D 夥伴角色
- 計算你的投資組合價值
- 從你第一次造訪開始追蹤經驗值

:::tip 手機用戶
IdleTrencher 完全支援手機！在手機瀏覽器中打開並連接你的手機錢包應用。你的夥伴們會隨時隨地等著你。
:::

## 隱私與安全

- 我們只讀取 **公開的區塊鏈數據** — 和任何區塊瀏覽器上能看到的資訊一樣
- 不收集個人數據
- 不追蹤或分析你的交易活動
- 隨時可以斷開錢包連接
