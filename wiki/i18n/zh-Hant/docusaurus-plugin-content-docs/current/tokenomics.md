# $IDLE 代幣

**$IDLE** 代幣驅動 IdleTrencher 生態系統。持有它可以解鎖更多功能並支持這個專案！

## 代幣資訊

| 屬性 | 值 |
|------|-----|
| **名稱** | IDLE |
| **鏈** | Solana |
| **發行** | PumpFun |
| **合約** | 待公布 |

## 分配

```
┌─────────────────────────────────────┐
│                                     │
│         90% 公開                    │
│     （流通供應量）                   │
│                                     │
├─────────────────────────────────────┤
│ 10% 團隊（黑客松儲備）               │
└─────────────────────────────────────┘
```

- **90%** — 透過 PumpFun 公開流通
- **10%** — 團隊分配用於黑客松和開發
