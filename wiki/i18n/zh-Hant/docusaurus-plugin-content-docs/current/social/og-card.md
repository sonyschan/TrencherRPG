# 可分享 OG 卡片

用精美的方式分享你的 IdleTrencher 投資組合！當你在社群媒體上分享錢包連結時，會自動產生漂亮的預覽卡片。

## 什麼是 OG 卡片？

OG（Open Graph）卡片是當你在 Twitter/X、Discord、Telegram 等社群媒體平台分享連結時出現的預覽圖片。IdleTrencher 會為你的錢包產生獨特且吸睛的卡片。

## 如何使用

1. **取得你的 OG 卡片 URL** - 使用你的錢包地址
2. **分享到任何地方** - Twitter、Discord、Telegram 等
3. **見證魔法** - 漂亮的預覽卡片會自動出現

### OG 圖片 URL

你的投資組合 OG 卡片可在以下網址取得：
```
https://idletrencher.xyz/api/og/你的錢包地址
```

例如：
```
https://idletrencher.xyz/api/og/9jwHJHSD7geYvTy6WUtoDVuuvuoJiWH2XHWMggPUpump
```

## 卡片特色

OG 卡片包含：

- **IdleTrencher 品牌設計** - 帶有 GM/GN 主題的藝術風格
- **動態錢包預覽** - 一目了然地展示你的投資組合
- **最佳化尺寸** - 適配所有主流平台（1200x630）
- **自動產生** - 無需手動截圖

## 主題

| 主題 | 說明 |
|------|------|
| **GM（早安）** | 明亮、活力的設計，適合白天分享 |
| **GN（晚安）** | 平靜、放鬆的設計，適合夜晚氛圍 |

## 分享技巧

1. **添加說明** - 告訴大家你在分享什麼：「來看看我的 IdleTrencher 夥伴們！」
2. **標記我們** - 提及 [@idleTrencher](https://x.com/idleTrencher) 有機會被精選展示
3. **使用標籤** - #IdleTrencher #Solana #CryptoGaming
