# 等級與經驗值系統

你的夥伴只要在你的投資組合中就會獲得 **經驗值 (EXP)**。持有越久，他們就越強！

![經驗值和等級](/img/partners-exp-level.jpg)

## 數學公式

IdleTrencher 使用透明、確定性的升級公式：

### 經驗值計算

```
有效經驗值 = 持有小時數 + 獎勵小時數
```

- **持有小時數**：從首次錢包連接以來的時間（1 小時 = 1 EXP）
- **獎勵小時數**：價格上漲帶來的額外經驗值（獎勵積極的市場參與）

### 等級公式

```
等級 = floor(sqrt(小時數 / 2.433))
```

這創造了 **平方根進度** — 早期等級來得很快，而更高等級需要指數級更多的時間。

### 每個等級所需時間

| 等級 | 所需小時數 | 大約時間 |
|------|------------|----------|
| 1 | 0 | 立即 |
| 2 | 10 | ~10 小時 |
| 5 | 61 | ~2.5 天 |
| 10 | 243 | ~10 天 |
| 20 | 973 | ~40 天 |
| 30 | 2,189 | ~3 個月 |
| 40 | 3,893 | ~5.5 個月 |
| 50 | 6,083 | ~8.5 個月 |
| 60 | 8,759 | ~1 年（最高）|

### 反向公式（從等級計算小時數）

```
小時數 = 等級² × 2.433
```

## 等級里程碑

| 等級 | 所需時間 |
|------|----------|
| Lv2 | ~10 小時 |
| Lv10 | ~10 天 |
| Lv20 | ~40 天 |
| Lv30 | ~3 個月 |
| Lv60 | ~1 年（最高）|

## 獎勵經驗值系統

價格上漲會給予 **獎勵小時數**，獎勵經歷正向市場波動的持有者：

- 當代幣價格上漲時，獎勵經驗值會累積
- 這創造了平衡策略：穩定持有獲得基礎成長，把握價格上漲獲得加速進度

:::warning 經驗值重置
如果夥伴從你的頂級持有中掉出，他們的經驗值會重置。這個設計鼓勵維持平衡的投資組合和一致的持有策略。
:::

## 為什麼這樣設計？

平方根公式確保：

1. **新玩家感受到進度** — 只要 10 小時就能達到 Lv2
2. **長期持有者獲得獎勵** — 更高等級展示真正的承諾
3. **最高等級可達成** — 約 1 年達到 Lv60 是現實的
4. **透明且可驗證** — 任何人都能計算預期等級

> 千里之行，始於持有一小時。🧘
