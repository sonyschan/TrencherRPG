# 探索其他錢包

有沒有好奇過你最喜歡的加密貨幣 KOL 實際上持有什麼？或者想用有趣的方式查看朋友的投資組合？

**探索模式** 讓你可以觀察任何公開的 Solana 錢包，看看他們的代幣變成什麼 3D 角色！

![Explore Mode](/img/explore.jpg)

## 如何使用

1. 點擊 **探索** 按鈕（放大鏡圖示）
2. 貼上任何 Solana 錢包地址
3. 看他們的投資組合活起來！

## 你可以看到什麼

探索別人的錢包時，你會看到：

- 🎭 他們的代幣夥伴變成 3D 角色
- 💰 大約的投資組合價值
- 📊 每個代幣的表現如何（跳舞 vs 運動）

:::note 隱私優先
我們只顯示 **公開的區塊鏈數據** — 和任何人在區塊瀏覽器上能看到的資訊一樣，只是看起來更有趣。沒有私人數據，沒有追蹤。
:::

## 有趣的使用案例

### 🕵️ 尋找 Alpha

看看聰明錢在持有什麼。如果他們的角色都在跳舞，也許他們發現了什麼...

### 👥 給朋友看

分享你的錢包地址，讓朋友用有格調的方式看你的投資組合。比發截圖酷多了！

### 📚 學習

加密貨幣新手？探索有經驗的交易者的錢包，看看分散投資的組合長什麼樣。
