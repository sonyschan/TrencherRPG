## 終極炫耀

當然，任何人都可以買代幣。但會交易、有眼光、能持有，三種能力加起來才是真功夫。排行榜將呈現出社群中代幣修為最高的持有者。

## 排名機制

你的 **總分** 是透過加總所有夥伴角色的等級來計算：

```
總分 = 所有夥伴等級相加
```

例如：
- 夥伴 1：Lv15
- 夥伴 2：Lv12
- 夥伴 3：Lv8
- **總分：35**

## 聯賽系統

不同資產等級在各自的聯賽中競爭 — 確保公平競爭：

| 聯賽 | 投資組合價值 |
|------|--------------|
| 銅牌 | < $1,000 |
| 銀牌 | $1,000 - $10,000 |
| 金牌 | $10,000 - $100,000 |
| 鑽石 | $100,000+ |

這樣，小戶和大戶各有自己的競爭空間。

## 為什麼要有排行榜？

說實話 — 加密貨幣可以很競爭。與其比誰交易最快或誰最早得到內線消息，我們想慶祝一種不同的成功：

**在持幣與積極交易當中，取得平衡的能力。**

聽起來很簡單，但在一個充滿百倍迷因幣和日常 rug pull 的世界裡，成為一個穩定的持有者本身就是一種成就。

> 「股市是一個把錢從沒耐心的人轉移到有耐心的人的工具。」
> — 華倫·巴菲特（他可能不懂迷因幣，但原則是對的）
