# 歡迎來到 IdleTrencher ✨

**IdleTrencher** 將您的 Solana 錢包轉化為直觀且生動的 3D 視覺化體驗。您的代幣持有將成為互動的夥伴角色，實時反映市場變化，讓投資組合的追蹤變得更具趣味與實用性。這是一款結合放置成長與 portfolio 視覺化的工具，適合希望平衡持有與市場參與的加密投資者與愛好者。

加密市場波動常帶來壓力。**IdleTrencher** 提供以下價值：

- 被動成長系統，獎勵持有與市場積極表現。
