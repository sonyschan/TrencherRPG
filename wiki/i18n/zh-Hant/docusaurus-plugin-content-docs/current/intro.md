# 歡迎來到 IdleTrencher ✨

**讓你的代幣錢包變得有趣、可愛！**

![IdleTrencher Banner](/img/banner.png)

## 為什麼選擇 IdleTrencher？

老實說 — 盯著加密貨幣的走勢圖真的很有壓力。價格不斷波動、FOMO、還有那些似乎永無止境的紅色日子... 😵‍💫

**如果查看投資組合可以變得...很放鬆呢？**

**IdleTrencher** 用視覺、遊戲化的創意方式為幣圈做改變。不用再盯著無聊的數字和讓人焦慮的圖表，你的代幣會變成可愛的 3D 角色，即時反應市場的變化。

- 📈 **代幣上漲？** 你的角色會開心地跳舞慶祝！
- 📉 **今天不太順利？** 他們正在做仰臥起坐，努力恢復中。
- 😌 **市場平靜？** 他們就悠閒地散步。

## 氛圍

IdleTrencher 是為 **加密貨幣愛好者** 設計的。無論你是鑽石手長期持有者，還是只是好奇朋友的投資組合，我們希望讓這個體驗：

- 🎮 **有趣** — 以你的持有代幣為角色的放置遊戲
- 😊 **放鬆** — 不再焦慮看圖，只有可愛的角色在悠閒活動
- 🎮 **好玩** — 遊戲化的升級、換皮膚、排行榜

> 「我以前每天因為壓力查看投資組合 50 次。現在我每天查看 50 次是因為我想看我的小夥伴們跳舞。」— *大概某個人說的*

## 開始使用

1. **連接你的錢包** — 我們支援 Phantom、Solflare 和其他 Solana 錢包
2. **看你的代幣活起來** — 你持有最多的代幣會變成 3D 夥伴
3. **讓他們升級** — 只要持有，你的夥伴就會獲得經驗值並變得更強

準備好認識你的夥伴了嗎？[**開始遊戲 →**](https://idletrencher.vercel.app)
