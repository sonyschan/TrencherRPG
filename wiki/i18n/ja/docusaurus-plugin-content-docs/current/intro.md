# IdleTrencherへようこそ ✨

**IdleTrencher** はあなたのSolanaウォレットを直感的でダイナミックな3Dビジュアル体験に変えます。トークン保有がインタラクティブなパートナーキャラクターになり、市場の変化をリアルタイムで反映し、ポートフォリオの追跡をより魅力的で実用的にします。放置成長とポートフォリオ可視化を組み合わせたツールで、保有と市場参加のバランスを取りたい暗号投資家向けです。

![IdleTrencher Banner](/img/banner.png)

## IdleTrencherの価値

暗号市場の変動はしばしばストレスをもたらします。**IdleTrencher** は以下の価値を提供します：

- トークンを3Dパートナーに変換し、アニメーションで価格変化を表現。
- 保有と積極的な市場パフォーマンスを報酬する受動的成長システム。
- 他の人の公開ウォレットを探索し、学習とインスピレーションの源に。
- オープンなコミュニティ貢献による継続的なコンテンツと機能の充実。

このプロジェクトは2026年Solanaハッカソンから生まれ、**$IDLE** をPumpFunで公平にローンチ — プレセールなし、インサイダー取引なし — 透明性と長期的な持続可能な発展にコミットしています。

## 雰囲気

- 🎮 **楽しく** — あなたのトークンがキャラクターになる放置ゲーム
- 😊 **リラックス** — チャートの不安はなく、可愛いキャラクターがのんびり
- 🎮 **面白い** — ゲーム化されたレベルアップ、スキン、ランキング

> 「以前はストレスで1日50回ポートフォリオをチェックしていました。今は小さな仲間たちのダンスを見たくて1日50回チェックしています。」— *たぶん誰か*

## 始め方

1. **ウォレットを接続** — Phantom、Solflare、その他のSolanaウォレットに対応
2. **トークンが動き出す** — 保有量の多いトークンが3Dパートナーに
3. **レベルアップさせよう** — 持っているだけでパートナーはEXPを獲得して強くなります
