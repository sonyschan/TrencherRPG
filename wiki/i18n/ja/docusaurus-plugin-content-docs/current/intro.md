# IdleTrencherへようこそ ✨

**あなたのトークンウォレットを楽しく、可愛く！**

![IdleTrencher Banner](/img/banner.png)

## なぜIdleTrencher？

正直に言うと — 仮想通貨のチャートを見続けるのはストレスです。絶え間ない価格変動、FOMO、終わりのない赤い日々... 😵‍💫

**ポートフォリオを確認するのが...リラックスできるとしたら？**

**IdleTrencher** は視覚的でゲーム性のあるクリエイティブな方法で仮想通貨を変えます。退屈な数字や不安を誘うチャートを見る代わりに、あなたのトークンがリアルタイムで市場に反応する可愛い3Dキャラクターになります。

- 📈 **トークンが上昇？** キャラクターが嬉しそうにダンスしてお祝い！
- 📉 **厳しい一日？** 腹筋運動をして回復に努めています。
- 😌 **市場が穏やか？** のんびりお散歩しています。

## 雰囲気

IdleTrencherは **仮想通貨を楽しむ人** のために設計されています。ダイヤモンドハンドの長期ホルダーでも、友達のポートフォリオが気になるだけでも、この体験を：

- 🎮 **楽しく** — あなたのトークンがキャラクターになる放置ゲーム
- 😊 **リラックス** — チャートの不安はなく、可愛いキャラクターがのんびり
- 🎮 **面白い** — ゲーム化されたレベルアップ、スキン、ランキング

> 「以前はストレスで1日50回ポートフォリオをチェックしていました。今は小さな仲間たちのダンスを見たくて1日50回チェックしています。」— *たぶん誰か*

## 始め方

1. **ウォレットを接続** — Phantom、Solflare、その他のSolanaウォレットに対応
2. **トークンが動き出す** — 保有量の多いトークンが3Dパートナーに
3. **レベルアップさせよう** — 持っているだけでパートナーはEXPを獲得して強くなります
