# レベル＆EXPシステム

パートナーはポートフォリオにいるだけで **経験値 (EXP)** を獲得します。長く保有するほど、強くなります！

![EXPとレベル](/img/partners-exp-level.jpg)

## 数学的公式

IdleTrencherは透明で決定論的なレベリング公式を使用しています：

### EXP計算

```
有効EXP = 保有時間（時間） + ボーナス時間
```

- **保有時間**：最初のウォレット接続からの時間（1時間 = 1 EXP）
- **ボーナス時間**：価格上昇による追加EXP（積極的な市場参加への報酬）

### レベル公式

```
レベル = floor(sqrt(時間 / 2.433))
```

これにより **平方根進行** が生まれます — 初期レベルは早く到達し、高レベルには指数的に多くの時間が必要です。

### 各レベルに必要な時間

| レベル | 必要時間 | おおよその期間 |
|--------|----------|----------------|
| 1 | 0 | 即時 |
| 2 | 10 | ~10時間 |
| 5 | 61 | ~2.5日 |
| 10 | 243 | ~10日 |
| 20 | 973 | ~40日 |
| 30 | 2,189 | ~3ヶ月 |
| 40 | 3,893 | ~5.5ヶ月 |
| 50 | 6,083 | ~8.5ヶ月 |
| 60 | 8,759 | ~1年（最大）|

### 逆公式（レベルから時間を計算）

```
時間 = レベル² × 2.433
```

## レベルマイルストーン

| レベル | 必要時間 |
|--------|----------|
| Lv2 | ~10時間 |
| Lv10 | ~10日 |
| Lv20 | ~40日 |
| Lv30 | ~3ヶ月 |
| Lv60 | ~1年（最大）|

## ボーナスEXPシステム

価格上昇は **ボーナス時間** を付与し、ポジティブな市場変動を経験したホルダーに報酬を与えます：

- トークン価格が上昇するとボーナスEXPが蓄積
- これによりバランスの取れた戦略が生まれます：安定保有で基本成長、価格上昇を捉えて加速的な進歩

:::warning EXPリセット
パートナーがトップ保有から外れると、EXPがリセットされます。この設計はバランスの取れたポートフォリオと一貫した保有戦略を促進します。
:::

## なぜこの設計？

平方根公式により：

1. **新規プレイヤーが進捗を感じる** — わずか10時間でLv2到達
2. **長期ホルダーが報われる** — 高レベルは真のコミットメントを示す
3. **最高レベルは達成可能** — 約1年でLv60は現実的
4. **透明で検証可能** — 誰でも期待レベルを計算できる
