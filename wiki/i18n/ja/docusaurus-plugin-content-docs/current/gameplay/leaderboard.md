## 究極の自慢

もちろん、誰でもトークンを買えます。でもトレードスキル、鋭い目、そして保有力 — この3つが揃ってこそ真のマスターです。ランキングでは、コミュニティで最もスキルの高いTrencherを紹介します。

## ランキングの仕組み

**総合スコア** は、すべてのパートナーキャラクターのレベルを合計して計算されます：

```
総合スコア = すべてのパートナーレベルの合計
```

例：
- パートナー1：Lv15
- パートナー2：Lv12
- パートナー3：Lv8
- **総合スコア：35**

### ボーナスメカニズム（設計中）

様々な実績を報酬するための追加スコアボーナスを設計中：

- 🎯 **多様性ボーナス** — 複数トークンにわたるバランスの取れたポートフォリオ
- 💎 **ダイヤモンドハンド** — 長期的な一貫した保有への追加ポイント
- 🚀 **成長ボーナス** — 最近レベルアップしたパートナーへの報酬
- 🎨 **スキンコレクション** — レアスキンのアンロックへのボーナス

*正確なボーナス計算式はローンチ前に発表されます。*

## リーグシステム

異なる資産ティアはそれぞれのリーグで競争 — 公平な競争を確保：

| リーグ | ポートフォリオ価値 |
|--------|-------------------|
| ブロンズ | < $1,000 |
| シルバー | $1,000 - $10,000 |
| ゴールド | $10,000 - $100,000 |
| ダイヤモンド | $100,000+ |

これにより、小口と大口それぞれが自分の競争空間を持てます。

## なぜランキング？

正直に言うと — 仮想通貨は競争的になりえます。誰が最速でトレードできるか、誰が最も早くインサイダー情報を得たかを競う代わりに、私たちは別の種類の成功を祝いたいと思います：

**保有と積極的なトレードのバランスを取る能力。**
