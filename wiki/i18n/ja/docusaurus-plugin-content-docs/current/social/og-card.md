# シェア可能なOGカード

IdleTrencherのポートフォリオをスタイリッシュにシェアしましょう！ソーシャルメディアでウォレットリンクをシェアすると、美しいプレビューカードが自動的に生成されます。

## OGカードとは？

OG（Open Graph）カードは、Twitter/X、Discord、Telegramなどのソーシャルメディアプラットフォームでリンクをシェアする際に表示されるプレビュー画像です。IdleTrencherはあなたのウォレット用にユニークで目を引くカードを生成します。

## 使い方

1. **ウォレットURLをコピー** - ブラウザのアドレスバーからコピー
2. **どこにでも貼り付け** - Twitter、Discord、Telegramなど
3. **魔法を見届けよう** - 美しいプレビューカードが自動的に表示されます

シェア可能なURL形式：
```
https://idletrencher.xyz/?wallet=あなたのウォレットアドレス
```

## カードの特徴

OGカードには以下が含まれます：

- **IdleTrencherブランドデザイン** - GM/GNテーマのアートワーク
- **動的ウォレットプレビュー** - ポートフォリオを一目で表示
- **最適化されたサイズ** - すべての主要プラットフォーム対応（1200x630）
- **自動生成** - スクリーンショット不要

## テーマ

| テーマ | 説明 |
|--------|------|
| **GM（グッドモーニング）** | 明るくエネルギッシュなデザイン、日中のシェアに最適 |
| **GN（グッドナイト）** | 穏やかでリラックスしたデザイン、夜の雰囲気に |

2. **タグ付け** - [@idleTrencher](https://x.com/idleTrencher) をメンションしてフィーチャーされるチャンスを

> 「パートナーたちがレベルアップ中！私の村に遊びに来てね 🏘️
> https://idletrencher.xyz/?wallet=...」

---

:::tip ヒント
OGカードはパフォーマンスのためにキャッシュされます。大きなポートフォリオの変更後にリフレッシュしたい場合は、`?v=2` のようなクエリパラメータを追加して強制的に再生成できます。
:::
