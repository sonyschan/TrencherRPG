# $IDLEトークン

**$IDLE** トークンはIdleTrencherエコシステムを支えています。保有することでより多くの機能をアンロックし、プロジェクトをサポートできます！

## トークン情報

| 項目 | 値 |
|------|-----|
| **名前** | IDLE |
| **チェーン** | Solana |
| **ローンチ** | PumpFun |
| **コントラクト** | 近日公開 |

## 配分

```
┌─────────────────────────────────────┐
│                                     │
│         90% パブリック              │
│     （流通供給量）                   │
│                                     │
├─────────────────────────────────────┤
│ 10% チーム（ハッカソン準備金）        │
└─────────────────────────────────────┘
```

- **90%** — PumpFunを通じてパブリックに流通
- **10%** — ハッカソンと開発用のチーム配分

## ユーティリティ

### 🎭 より多くのパートナー

$IDLEを多く保有するほど、より多くのトークンパートナーを持てます：

| $IDLE保有量 | パートナー数 |
|-------------|--------------|
| 0 | 1パートナー（無料枠）|
| 10,000 | 2パートナー |
| 20,000 | 3パートナー |
| 30,000 | 4パートナー |
| ... | 10kごとに+1 |
| 90,000+ | 10パートナー（上限）|

### 🎨 プレミアム機能

今後の$IDLEユーティリティには以下が含まれる可能性があります：
- 限定スキンのアンロック
- 新機能への優先アクセス
- ガバナンス投票権

## フェアローンチ

IdleTrencherはハッカソン中に構築され、仮想通貨をみんなにとってもっと楽しくすることを目標としています。PumpFunを通じて$IDLEをローンチし、公平で透明な配分を確保します。

**プレセールなし。インサイダー取引なし。ただ良い雰囲気。**
