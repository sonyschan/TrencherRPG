# はじめに

数秒でウォレットを接続して、IdleTrencherの旅を始めましょう！

![ウォレット接続](/img/wallet-connection.jpg)

## ウォレット接続

IdleTrencherは **Privy** — 信頼できるサードパーティ認証システム — を使用して、Solanaウォレットを安全に接続します。これは：

- **シードフレーズの露出なし** — 私たちは秘密鍵を要求したり保存したりしません
- **複数ウォレット対応** — Phantom、Solflare、その他のSolanaウォレットで接続可能
- **シームレスな体験** — デスクトップとモバイルの両方でワンクリック接続

## 接続方法

1. **ウォレット接続** ボタンをクリック
2. Privyモーダルからお好みのウォレットを選択
3. ウォレットで接続を承認
4. トークンがパートナーに変身！

## 次に何が起こる？

接続後、IdleTrencherは自動的に：

- ウォレットのトークン保有をスキャン
- 最も多く保有しているトークンを3Dパートナーキャラクターに変換
- ポートフォリオ価値を計算
- 初回訪問からEXPの追跡を開始

:::tip モバイルユーザー
IdleTrencherは完全モバイル対応！スマホのブラウザで開いて、モバイルウォレットアプリに接続しましょう。パートナーたちはどこでもあなたを待っています。
:::

## プライバシーとセキュリティ

- **パブリックなブロックチェーンデータ** のみを読み取ります — ブロックエクスプローラーで誰でも見られる情報と同じです
- 個人データの収集なし
- 取引活動の追跡や分析なし
- いつでもウォレット接続を解除可能
