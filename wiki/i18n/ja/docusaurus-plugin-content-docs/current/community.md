## 3Dキャラクターを貢献

キャラクターコレクションを拡大するために **3Dアーティスト** を積極的に募集しています！

### なぜ貢献？

✅ あなたの作品がゲームの一部になります
✅ 自分のスキンを自動的に使用できます
✅ ゲームとWikiでクレジット
✅ 楽しいコミュニティプロジェクトに参加

### キャラクター要件

アニメーションシステムとの互換性を確保するため、貢献するキャラクターは：

| 要件 | 詳細 |
|------|------|
| **フォーマット** | GLTF/GLB形式 |
| **リギング** | Mixamoまたは類似システムと互換性 |
| **アニメーション** | 歩く、走る、お祝い、腹筋/エクササイズ |
| **スタイル** | ローポリ、スタイライズド（現在のアートスタイルに合わせて）|
| **コンテンツ** | 全年齢対応、不適切な画像なし |

### 提出方法

1. [GitHubリポジトリ](https://github.com/sonyschan/TrencherRPG)を **Fork**
2. キャラクターを `/public/assets/characters/` に追加
3. プレビュー画像を含める
4. **Pull Request** を提出して以下を記載：
   - キャラクター名
   - 簡単な説明
   - クレジット名（または匿名）

提出をレビューして、承認されたキャラクターをゲームに統合します！
